# Number Guessing Game (C)

A simple console based number guessing game written in C.

## Features
- Random number generation
- User input
- Hint system (Too high / Too low)
- Score tracking

## How to Run

1. Compile the program

gcc game.c -o game

2. Run the program

./game

## Example Output

Enter your guess: 50
Too high!

Enter your guess: 25
Too low!

Enter your guess: 37
Correct!
